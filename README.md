# MovieSuggest
A Ionic v6 project to recommend movies and series to friends


## Initialize Ionic 6
`npm install @ionic/angular@6`
`npm install -g @ionic/cli`


## Start an app
`ionic start`


## Run the app
`ionic serve`


## Firebase
`npm i firebase @angular/fire`
`npm i -g firebase-tools`
`firebase login`
`firebase init`
`ionic build --prod --release`
`firebase deploy`
